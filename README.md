# Inventory App

## Overview
This project is a Django-based inventory management system. Users can register, add products, and manage an inventory system. After logging in, users can see and interact with their products.

## Features
- User authentication (login, register)
- Product CRUD functionality (Create, Read, Update, Delete)
- View products on an organized table
- Responsive user interface with Bootstrap
